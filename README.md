# Contribute
first clone the repo
```
git clone https://github.com/usertermed/rockalmighty.git
cd rockalmighty
npm install
```
then rename .env.example to .env.local and add your openai api key
```
cp .env.example .env.local
# (add your openai api key to .env.local)
```
then run the dev server
```
npm run dev
```
